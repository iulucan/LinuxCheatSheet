Linux Command Cheat Sheet

File Operations

ls – List directory contents

cd [dir] – Change directory

pwd – Show current directory

cp [src] [dst] – Copy file or directory

mv [src] [dst] – Move/rename file or directory

rm [file] – Delete file

rm -r [dir] – Delete directory recursively

touch [file] – Create empty file

mkdir [dir] – Create directory

File Viewing

cat [file] – View file contents

less [file] – View file with paging

head [file] – Show first lines

tail [file] – Show last lines

tail -f [file] – Live monitor file

Permissions

chmod [mode] [file] – Change file permissions

chown [user]:[group] [file] – Change owner

ls -l – Show permissions

Searching

find [dir] -name "[pattern]" – Search for files

grep "[text]" [file] – Search text in file

grep -r "[text]" [dir] – Recursive grep

System Info

uname -a – System info

top – Running processes

htop – Interactive process viewer

df -h – Disk usage

free -h – Memory usage

uptime – System uptime

Networking

ip a – Show IP addresses

ping [host] – Ping host

netstat -tuln – Show ports

curl [url] – Make web request

wget [url] – Download file

Package Management (Debian/Ubuntu)

sudo apt update – Refresh package list

sudo apt install [pkg] – Install package

sudo apt remove [pkg] – Remove package

Package Management (RedHat/CentOS)

sudo yum install [pkg] – Install package

sudo yum remove [pkg] – Remove package

Others

man [cmd] – Manual for command

history – Show command history

alias ll='ls -la' – Create shortcut

sudo [cmd] – Run as superuser
